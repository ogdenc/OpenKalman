Description

template<typename Coeffs, typename NestedMatrix>
struct OpenKalman::Mean< Coeffs, NestedMatrix >

A set of one or more column vectors, each representing a statistical mean.

A typed vector.

Unlike OpenKalman::Matrix, the columns of a Mean are untyped. When a Mean is converted to an OpenKalman::Matrix, the columns are assigned type Axis. Example declaration: Mean<Coefficients<Axis, Axis, angle::Radians>, 1, native_matrix_t<double, 3, 1>> x; This declares a 3-dimensional vector x, where the coefficients are, respectively, an Axis, an Axis, and an angle::Radians, all of scalar type double. The underlying representation is an Eigen3 column vector.

Template Parameters
CoefficientsCoefficient types of the mean (e.g., Axis, Polar).
NestedMatrixThe underlying native matrix or matrix expression.
