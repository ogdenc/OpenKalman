173  * \internal
174  * \brief An array of functions that convert the coefficients to coordinates in Euclidean space.
175  * \details The functions in the array take the coefficients and convert them to
176  * Cartesian coordinates in a Euclidean space, depending on the type of each coordinate.
177  * Each array element is a function taking a ''get coefficient'' function and returning a coordinate value.
178  * The ''get coefficient'' function takes the index of a column within a row vector and returns the coefficient.
179  * \note This should be accessed only through \ref to_euclidean_coeff.
180  * \tparam Scalar The scalar type (e.g., double).
181  * \tparam i The index of the first spherical coefficient that is being transformed.
