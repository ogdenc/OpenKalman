Description

template<typename RowCoefficients, typename ColumnCoefficients, typename NestedMatrix>
struct OpenKalman::Matrix< RowCoefficients, ColumnCoefficients, NestedMatrix >

It is a wrapper for a native matrix type from a supported matrix library such as Eigen. The matrix can be thought of as a transformation from X to Y, where the coefficients for each of X and Y are typed. Example declarations:

  • Matrix<Coefficients<Axis, Axis, angle::Radians>, Coefficients<Axis, Axis>, native_matrix_t<native_matrix_t<double, 3, 2>> x;
  • Matrix<double, Coefficients<Axis, Axis, angle::Radians>, Coefficients<Axis, Axis>, native_matrix_t<double, 3, 2>> x;
    Template Parameters
    RowCoefficientsA set of coefficients (e.g., Axis, Spherical, etc.) corresponding to the rows.
    ColumnCoefficientsAnother set of coefficients corresponding to the columns.
    NestedMatrixThe underlying native matrix or matrix expression.
