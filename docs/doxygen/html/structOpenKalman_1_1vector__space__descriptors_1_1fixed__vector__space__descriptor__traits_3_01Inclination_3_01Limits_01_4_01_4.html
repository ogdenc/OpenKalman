Description

template<typename Limits = inclination::RadiansLimits>
struct OpenKalman::vector_space_descriptors::fixed_vector_space_descriptor_traits< Inclination< Limits > >

φdown≤φ≤φup, where φdown is a real number representing down, and φup is a real number representing up. Normally, the horizon will be zero and φdown=−φup, but in general, the horizon is at ½(φdown+−φup). The inclinations inclination::Radians and inclination::Degrees are predefined.

Template Parameters
LimitsA class template defining the real values down and up, where down=φdown and up=φup. Scalar is a scalar_type.

The inclination angle corresponds to x and y coordinates in quadrants I or IV of a unit circle. By convention, the minimum angle limit Limits<Scalar::min corresponds to the point (-1,0) in Euclidean space, and the angle is scaled so that the difference between Limits<Scalar>::min and Limits<<Scalar>::max is 2π, so Limits<Scalar>::max wraps back to the point (-1, 0).

Parameters
euclidean_local_indexA local index accessing either the x (if 0) or y (if 1) coordinate in Euclidean space
