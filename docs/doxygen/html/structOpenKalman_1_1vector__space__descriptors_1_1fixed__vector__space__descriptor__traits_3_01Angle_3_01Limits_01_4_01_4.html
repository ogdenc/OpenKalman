template<typename Limits = angle::RadiansLimits>
struct OpenKalman::vector_space_descriptors::fixed_vector_space_descriptor_traits< Angle< Limits > >

An angle wraps to a given interval [max,min) when it increases or decreases outside that range. There are several predefined angles, including angle::Radians, angle::Degrees, angle::PositiveRadians, angle::PositiveDegrees, and angle::Circle. See David Frederic Crouse, Cubature/Unscented/Sigma Point Kalman Filtering with Angular Measurement Models, 18th Int'l Conf. on Information Fusion 1550, 1553 (2015).

Template Parameters
LimitsA template class defining the real values min and max, representing minimum and maximum values, respectively, beyond which wrapping occurs. This range must include 0.
