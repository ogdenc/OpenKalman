Description

template<typename NestedMatrix, TriangleType storage_triangle>
struct OpenKalman::Eigen3::SelfAdjointMatrix< NestedMatrix, storage_triangle >

Template Parameters
NestedMatrixThe Eigen matrix on which the self-adjoint matrix is based.
storage_triangleThe triangle (TriangleType::upper or TriangleType::lower) in which the data is stored. The other triangle is not used, and coefficients are mapped from the other triangle to ensure that the matrix is always self-adjoint.
