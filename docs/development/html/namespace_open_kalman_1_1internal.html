◆ __pad2__

noexcept OpenKalman::internal::__pad2__

This overload is operative if Arg's nested matrix is the same as its cholesky nested matrix as well as NestedMatrix. This only occurs when constructing the square or square_root of a matrix. In other words, Arg is a Case 1 or Case 2 covariance.
