Description

template<typename NestedMatrix>
struct OpenKalman::Eigen3::DiagonalMatrix< NestedMatrix >

The matrix is guaranteed to be diagonal. It is self_contained iff NestedMatrix is self_contained. Implicit conversions are available from any diagonal_matrix of compatible size.

Template Parameters
NestedMatrixA column_vector expression defining the diagonal elements. Elements outside the diagonal are automatically 0.
Note
This has the same name as Eigen::DiagonalMatrix, and is intended as a replacement.
