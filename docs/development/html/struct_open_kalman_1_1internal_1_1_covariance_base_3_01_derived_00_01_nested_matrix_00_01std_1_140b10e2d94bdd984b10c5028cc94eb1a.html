template<typename Derived, typename NestedMatrix>
struct OpenKalman::internal::CovarianceBase< Derived, NestedMatrix, std::enable_if_t< case1or2< Derived, NestedMatrix > and(not self_contained< NestedMatrix >)> >

This specialization is operative if NestedMatrix is not self-contained and either

Derived is not a square root and NestedMatrix is self-adjoint; or

Derived is a square root and NestedMatrix is triangular.

In this case, NestedMatrix and the cholesky nested matrix are the same.
