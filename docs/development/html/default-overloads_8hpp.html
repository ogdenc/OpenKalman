◆ OPENKALMAN_FIRST_INTERFACE

#define OPENKALMAN_FIRST_INTERFACE

The first-defined interface will define defaults such as the default nested matrices for library classes like Matrix, and the deduction guides for constructors that take only scalar coefficients.
