OpenKalman::Eigen3::FromEuclideanExpr< Coefficients, NestedMatrix > Struct Template Reference

Description

template<typename Coefficients, typename NestedMatrix>
struct OpenKalman::Eigen3::FromEuclideanExpr< Coefficients, NestedMatrix >

An expression that transforms angular or other modular coefficients back from Euclidean space.

This is the counterpart expression to ToEuclideanExpr. FromEuclideanExpr<C, ToEuclideanExpr<C, M>> acts to wrap the angular/modular values in M.

Template Parameters
CoefficientsThe coefficient types.
NestedMatrixThe pre-transformed column vector, or set of column vectors in the form of a matrix.
