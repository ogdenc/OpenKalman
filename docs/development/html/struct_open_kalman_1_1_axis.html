OpenKalman::Axis Struct Reference

A real or integral number, (−∞,∞). More...

#include <coefficient_forward-declarations.hpp>

Description

This is the default coefficient type. No wrapping occurs, and matrices operate as usual.

See also the following functions for accessing coefficient properties:

  • internal::to_euclidean_coeff: Get a coordinate in Euclidean space corresponding to a coefficient in a matrix with typed coefficients.
  • internal::from_euclidean_coeff: Get a typed coefficient corresponding to its corresponding coordinate in Euclidean space.
  • internal::wrap_get: Wrap a given coefficient and return its wrapped, scalar value.
  • internal::wrap_set Set the scalar value of a given typed coefficient in a matrix, and wrap the matrix column.
