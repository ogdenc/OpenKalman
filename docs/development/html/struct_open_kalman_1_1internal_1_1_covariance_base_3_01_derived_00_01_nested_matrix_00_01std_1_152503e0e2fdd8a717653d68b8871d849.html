template<typename Derived, typename NestedMatrix>
struct OpenKalman::internal::CovarianceBase< Derived, NestedMatrix, std::enable_if_t<(not case1or2< Derived, NestedMatrix >) and self_contained< NestedMatrix > > >

This specialization is operative if ArgType is self-contained and

Derived is a square root or NestedMatrix is not self-adjoint; and

Derived is not a square root or NestedMatrix is not triangular.

In this case, NestedMatrix and the cholesky nested matrix are different.
