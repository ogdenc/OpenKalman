OpenKalman::Eigen3::ToEuclideanExpr< Coefficients, NestedMatrix > Struct Template Reference

Description

template<typename Coefficients, typename NestedMatrix>
struct OpenKalman::Eigen3::ToEuclideanExpr< Coefficients, NestedMatrix >

An expression that transforms coefficients into Euclidean space for proper wrapping.

This is the counterpart expression to FromEuclideanExpr. FromEuclideanExpr<C, ToEuclideanExpr<C, M>> acts to wrap the angular/modular values in M.

Template Parameters
CoefficientsThe coefficient types.
NestedMatrixThe pre-transformed column vector, or set of column vectors in the form of a matrix.
