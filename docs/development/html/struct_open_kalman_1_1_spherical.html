Description

template<typename C1, typename C2, typename C3>
struct OpenKalman::Spherical< C1, C2, C3 >

Coefficient1, Coefficient2, and Coefficient3 must be some combination of Distance, Inclination, and Angle in any order, reflecting the distance, inclination, and azimuth, respectively. Spherical coordinates span three adjacent coefficients in a matrix.

Examples
Spherical<Distance, inclination::Degrees, angle::Radians>,
Spherical<angle::PositiveDegrees, Distance, inclination::Radians>
Template Parameters
C1,C2,C3Distance, inclination, and Angle, in any order. By default, they are Distance, angle::Radians, and inclination::Radians, respectively.

See also the following functions for accessing coefficient properties:

  • internal::to_euclidean_coeff: Get a coordinate in Euclidean space corresponding to a coefficient in a matrix with typed coefficients.
  • internal::from_euclidean_coeff: Get a typed coefficient corresponding to its corresponding coordinate in Euclidean space.
  • internal::wrap_get: Wrap a given coefficient and return its wrapped, scalar value.
  • internal::wrap_set Set the scalar value of a given typed coefficient in a matrix, and wrap the matrix column.
