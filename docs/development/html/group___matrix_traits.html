MatrixTraits

Detailed Description

A type trait class for any matrix T.

Template Parameters
TThe matrix type. The type is treated as non-qualified, even if it is const or a reference. This class includes key information about a matrix or matrix expression, such as its dimensions, coefficient types, etc.

Public Static Attributes

static constexpr std::size_t dimensionThe number of rows in the matrix.
static constexpr std::size_t columnsThe number of columns in the matrix.

Public Aliases

using NestedMatrix

(optional) If T has a nested matrix, this is an alias for that nested matrix.

using Scalar

The scalar type of T.

using SelfContained

A self_contained type equivalent to T.

using RowCoefficients

The coefficient types associated with the rows of T. This is only applicable for matrices with typed coefficients.

using ColumnCoefficients

The coefficient types associated with the columns of T. This is only applicable for matrices with typed coefficients.

template<std::size_t rows = dimension, std::size_t cols = columns, typename S = Scalar>
using NativeMatrix

A writable, native matrix type equivalent in size and shape to this matrix by default. Alternatively, you can specify the rows, columns, and Scalar type of the new matrix.

template<TriangleType storage_triangle = TriangleType::lower, std::size_t dim = dimension, typename S = Scalar>
using SelfAdjointBaseType

A writable, native self-adjoint matrix type equivalent to this matrix. Alternatively, you can specify the storage_triangle (upper, lower, diagonal) where the coefficients are stored, the dimension of the matrix, and the Scalar type (integral or floating-point) of the new matrix.

template<TriangleType triangle_type = TriangleType::lower, std::size_t dim = dimension, typename S = Scalar>
using TriangularBaseType

A writable, native truangular matrix type equivalent to this matrix. Alternatively, you can specify the triangle_type (upper, lower, diagonal) of the triangular matrix, the dimension of the matrix, and the Scalar type (integral or floating-point) of the new matrix.

Public Static Member Functions

template<typename Arg>
static auto make(Arg&& arg) noexcept

(If T is nestable in T) Make a matrix of type T from a native matrix. It might have a different size and shape.

static auto zero()

Make a matrix of type T with only zero coefficients.

static auto identity()

Make an identity matrix based on T. The resulting type will be a square matrix of dimension MatrixTraits<T>::dimension.
