OpenKalman::Eigen3::ConstantMatrix< Scalar, constant, rows_, columns > Struct Template Reference

A matrix in which all elements are a constant at compile time. More...

Description

template<typename Scalar, auto constant, std::size_t rows_, std::size_t columns>
requires (rows_ > 0) and (columns > 0)
struct OpenKalman::Eigen3::ConstantMatrix< Scalar, constant, rows_, columns >
Note
This is necessary because Eigen3 types do not provide for compile-time constant matrices.
Template Parameters
ScalarThe scalar type.
rowsThe number of rows (0 if dynamic_rows).
columnsThe number of columns (0 if dynamic_columns).
