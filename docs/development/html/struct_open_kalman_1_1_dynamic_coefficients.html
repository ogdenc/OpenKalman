Description

template<typename Scalar_>
struct OpenKalman::DynamicCoefficients< Scalar_ >

At compile time, the structure is treated if it has zero dimensions.

See also the following functions for accessing coefficient properties:

  • internal::to_euclidean_coeff(Coeffs&& coeffs, const std::size_t row, const F& get_coeff): Get a coordinate in Euclidean space corresponding to a coefficient in a matrix with typed coefficients.
  • internal::from_euclidean_coeff(Coeffs&& coeffs, const std::size_t row, const F& get_coeff): Get a typed coefficient corresponding to its corresponding coordinate in Euclidean space.
  • internal::wrap_get(Coeffs&& coeffs, const std::size_t row, const F& get_coeff): Wrap a given coefficient and return its wrapped, scalar value.
  • internal::wrap_set(Coeffs&& coeffs, const std::size_t row, const Scalar s, const FS& set_coeff, const FG& get_coeff)
