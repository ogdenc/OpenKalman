template<template< typename Scalar > typename Limits = angle::limits::Radians>
struct OpenKalman::Angle< Limits >

An angle or any other simple modular value.

An angle wraps to a given interval [max,min) when it increases or decreases outside that range. There are several predefined angles, including angle::Radians, angle::Degrees, angle::PositiveRadians, angle::PositiveDegrees, and angle::Circle.

Template Parameters
LimitsA class template defining the real values min and max, representing minimum and maximum values, respectively, beyond which wrapping occurs. Scalar is a std::floating_point type.

An array of functions (here, just one) that convert coordinates in Euclidean space into an angle.

The function in the array takes x and y coordinates representing a location on a unit circle, and convert those coordinates to an angle. The array element is a function taking a ''get coefficient'' function and returning an angle. The ''get coefficient'' function takes the index of a column within a row vector and returns either x (index i) or y (index i+1).

Each void function in the array takes a scalar value and ''set coefficient'' function. The ''set coefficient'' function takes a scalar value and an index of a column within a row vector and sets the coefficient at that index to a wrapped version of the scalar input.

The type of the result when subtracting two Angle values.

A distance between two points on a circle cannot be more than the circumference of the circle, so it must be wrapped as an Angle. See David Frederic Crouse, Cubature/Unscented/Sigma Point Kalman Filtering with Angular Measurement Models, 18th Int'l Conf. on Information Fusion 1550, 1553 (2015).
