Description

template<template< typename Scalar > typename Limits = angle::limits::Radians>
struct OpenKalman::Angle< Limits >

An angle wraps to a given interval [max,min) when it increases or decreases outside that range. There are several predefined angles, including angle::Radians, angle::Degrees, angle::PositiveRadians, angle::PositiveDegrees, and angle::Circle.

Template Parameters
LimitsA class template defining the real values min and max, representing minimum and maximum values, respectively, beyond which wrapping occurs. Scalar is a std::floating_point type.

See also the following functions for accessing coefficient properties:

  • internal::to_euclidean_coeff: Get a coordinate in Euclidean space corresponding to a coefficient in a matrix with typed coefficients.
  • internal::from_euclidean_coeff: Get a typed coefficient corresponding to its corresponding coordinate in Euclidean space.
  • internal::wrap_get: Wrap a given coefficient and return its wrapped, scalar value.
  • internal::wrap_set Set the scalar value of a given typed coefficient in a matrix, and wrap the matrix column.
