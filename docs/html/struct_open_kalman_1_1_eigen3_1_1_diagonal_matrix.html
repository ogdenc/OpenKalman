template<typename BaseMatrix>
struct OpenKalman::Eigen3::DiagonalMatrix< BaseMatrix >

A diagonal matrix, based on an Eigen matrix.

Note
This has the same name as Eigen::DiagonalMatrix, and is intended as an improved replacement.
