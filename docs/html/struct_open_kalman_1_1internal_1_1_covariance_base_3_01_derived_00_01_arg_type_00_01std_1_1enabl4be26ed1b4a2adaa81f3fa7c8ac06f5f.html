Detailed Description

template<typename Derived, typename ArgType>
struct OpenKalman::internal::CovarianceBase< Derived, ArgType, std::enable_if_t<((is_self_adjoint_v< ArgType > and not is_square_root_v< Derived >) or(is_triangular_v< ArgType > and is_square_root_v< Derived >)) and not(std::is_lvalue_reference_v< ArgType > or std::is_lvalue_reference_v< typename MatrixTraits< ArgType >::BaseMatrix >)> >

Base of Covariance and SquareRootCovariance classes, if ArgType is not an lvalue reference. No conversion is necessary if either (1) Derived is not a square root and the base is self-adjoint; or (2) Derived is a square root and the base is triangular.
