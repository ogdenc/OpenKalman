An array of functions that convert coordinates in Euclidean space into the typed coordinates.

The functions in the array take Cartesian coordinates, and convert them to the typed coordinates. The array element is a function taking a ''get coefficient'' function and returning the typed coordinates. The ''get coefficient'' function takes the index of a column within a row vector and returns one of the Cartesian coordinates.

Template Parameters
ScalarThe scalar type (e.g., double).
iThe index of the first of the Cartesian coordinates being transformed back to their respective types. This may be non-zero if this set of coefficients is part of a larger set of composite coordinates.

An array of functions that convert the coefficients to coordinates in Euclidean space.

The functions in the array take the coefficients and convert them to Cartesian coordinates in a Euclidean space, depending on the type of each coordinate. Each array element is a function taking a ''get coefficient'' function and returning a coordinate value. The ''get coefficient'' function takes the index of a column within a row vector and returns the coefficient.

Template Parameters
ScalarThe scalar type (e.g., double).
iThe index of the first spherical coefficient that is being transformed. This may be non-zero if this set of coefficients is part of a larger set of composite coordinates.

An array of functions that wraps and sets an existing matrix coefficient.

Each void function in the array takes a scalar value and ''set coefficient'' function. The ''set coefficient'' function takes a scalar value and an index of a column within a row vector and sets the coefficient at that index to a wrapped version of the scalar input.
