Detailed Description

template<typename RowCoefficients, typename ColumnCoefficients, typename BaseMatrix>
struct OpenKalman::TypedMatrix< RowCoefficients, ColumnCoefficients, BaseMatrix >

A typed matrix.

A matrix representing a transformation from X to Y with typed coefficients for each of X and Y. Example declaration: TypedMatrix<double, Coefficients<Axis, Axis, Angle>, Coefficients<Axis, Axis>> x;
