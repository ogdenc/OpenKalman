template<typename Traits = AngleTraits>
using OpenKalman::Inclination< Traits >::difference_type = Coefficients<Axis>

Because Inclination is limited to ±π/2, a difference between two Inclinations does not wrap, and is treated as Axis. See David Frederic Crouse, Cubature/Unscented/Sigma Point Kalman Filtering with Angular Measurement Models, 18th Int'l Conf. on Information Fusion 1550, 1555 (2015).
