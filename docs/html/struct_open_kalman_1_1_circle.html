template<typename Traits >
using OpenKalman::Circle< Traits >::difference_type = Coefficients<Circle>

A distance between to points on a circle cannot be more than the circumference of the circle, so it must be wrapped as Circle. See David Frederic Crouse, Cubature/Unscented/Sigma Point Kalman Filtering with Angular Measurement Models, 18th Int'l Conf. on Information Fusion 1550, 1553 (2015).
