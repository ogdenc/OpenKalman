◆ difference_type

using OpenKalman::Distance::difference_type = Axis

The type of the result when subtracting two Distance values.

A difference between two distances can be positive or negative, and is treated as Axis. See David Frederic Crouse, Cubature/Unscented/Sigma Point Kalman Filtering with Angular Measurement Models, 18th Int'l Conf. on Information Fusion 1553, 1555 (2015).

An array of functions (here, just one) that transform a coordinate in Euclidean space into a distance.

The resulting distance is always positive, so the functions take the absolute value. The array element is a function taking a ''get coefficient'' function and returning a positive distance. The ''get coefficient'' function takes the index of a column within a row vector and returns the value.

An array of functions (here, just one) that transform an distance coefficient to Euclidean space.

Because a distance already represents a point in Euclidean space, this is an identity function. Each array element is a function taking a ''get coefficient'' function and returning a coordinate value. The ''get coefficient'' function takes the index of a column within a row vector and returns the coefficient.

An array of functions (here, just one) that sets a matrix coefficient to a wrapped distance.

Each void function in the array takes a scalar value and ''set coefficient'' function. The ''set coefficient'' function takes a scalar value and an index of a column within a row vector and sets the coefficient at that index to the absolute value of that scalar input.
