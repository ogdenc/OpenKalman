Detailed Description

template<typename Coeffs, typename BaseMatrix>
struct OpenKalman::Mean< Coeffs, BaseMatrix >

A typed vector.

A set of column vectors representing one or more means. Generally, a column vector representing a mean. Alternatively, it can be a 2D matrix representing a collection of column vectors of the same coefficient types, each column vector representing a distinct mean. Example declaration: Mean<Coefficients<Axis, Axis, Angle>, 1, Eigen::Matrix<double, 3, 1>> x; This declares a 3-dimensional vector x, where the coefficients are, respectively, an Axis, an Axis, and an Angle, all of scalar type double. The underlying representation is an Eigen3 column vector.

By default when using Eigen3, a Mean is an Eigen3 column vector corresponding to the Coefficients.

Template Parameters
CoefficientsCoefficient types of the mean (e.g., Axis, Polar).
BaseMatrixRegular matrix on which the mean is based (usually a column vector).
