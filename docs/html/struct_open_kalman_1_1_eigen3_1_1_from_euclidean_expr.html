OpenKalman::Eigen3::FromEuclideanExpr< Coefficients, BaseMatrix > Struct Template Reference

Description

template<typename Coefficients, typename BaseMatrix>
struct OpenKalman::Eigen3::FromEuclideanExpr< Coefficients, BaseMatrix >

An expression that transforms angular or other modular coefficients back from Euclidean space.

This is the counterpart expression to ToEuclideanExpr. FromEuclideanExpr<C, ToEuclideanExpr<C, B>> acts to wrap the angular/modular values in B.

Template Parameters
CoefficientsThe coefficient types.
BaseMatrixThe pre-transformed column vector, or set of column vectors in the form of a matrix.
