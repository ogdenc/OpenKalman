An array of functions (here, just one) that transform a coordinate in Euclidean space into an axis.

Because an axis already represents a point in Euclidean space, this is an identity function. The array element is a function taking a ''get coefficient'' function and returning an axis. The ''get coefficient'' function takes the index of a column within a row vector and returns the value.

Template Parameters

iThe index of the coefficient that is being transformed.

An array of functions (here, just one) that sets a matrix coefficient to an axis value.

Each void function in the array takes a scalar value and ''set coefficient'' function. The ''set coefficient'' function takes a scalar value and an index of a column within a row vector and sets the coefficient at that index to that (unchanged) scalar input.
