Description

template<typename RowCoefficients, typename ColumnCoefficients, typename BaseMatrix>
struct OpenKalman::Matrix< RowCoefficients, ColumnCoefficients, BaseMatrix >

A matrix with typed rows and columns.

It is a wrapper for a native matrix type from a supported matrix library such as Eigen. The matrix can be thought of as a transformation from X to Y, where the coefficients for each of X and Y are typed. Example declaration: Matrix<double, Coefficients<Axis, Axis, angle::Radians>, Coefficients<Axis, Axis>, Eigen::Matrix<double, 3, 2>> x;

Template Parameters
RowCoefficientsA set of coefficients (e.g., Axis, Spherical, etc.) corresponding to the rows.
ColumnCoefficientsAnother set of coefficients corresponding to the columns.
NativeMatrixThe underlying native matrix or matrix expression.
